Random Number Generator

This function takes a minimum and maximum value and returns a random integer that is between those values.

Named Function

            function(min,max){
                return min + Math.random() * (max - min);
            }

Function Expression

            const random = function(min,max) {
                return min + Math.random() * (max - min);
            }

Arrow Function

            const random = (min,max) => min + Math.random() * (max - min);
