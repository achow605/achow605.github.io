Rules:
1. Two players will each start with 5 cards
2. In each turn, a player can place a card
3. The played card must match either the color or card type
4. If there are no matches, the player will draw a card and pass
5. If you have more than 8 cards, your top card will be replaced!
6. First player to one card wins!
